# Train-test split

Train-test split

## Usage

``` r
split_data(dat, p_test = 0.25)
```

## Arguments

- dat:

  A
  [`JointData`](https://generable.github.io/bmstate/reference/JointData.md)
  object

- p_test:

  Proportion of test subjects

## Value

A list with test and train
[`JointData`](https://generable.github.io/bmstate/reference/JointData.md)
objects
