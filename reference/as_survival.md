# PathData to event-free survival format

PathData to event-free survival format

## Usage

``` r
as_survival(pd, event)
```

## Arguments

- pd:

  A
  [`PathData`](https://generable.github.io/bmstate/reference/PathData.md)
  object

- event:

  Name of the state corresponding to the event of interest (character)

## Value

A
[`PathData`](https://generable.github.io/bmstate/reference/PathData.md)
object
